배열(리스트)에서 요소찾기

arr = ["복숭아~","오렌쥐~","샤인머스캣","사과","복숭아~","메론","참외","오렌쥐~", "복숭아~"]; 0,4,8 = 복숭아가 겹침 1,7 = 오렌쥐가 겹침 indexOf(item, start) : item의 위치를 찾는다 (겹치면 첫번째 찾는 것을 반환)
lastIndexOf(item, start) : 마지막 item의 위치를 찾는다

배열의 추가 /삭제

arr = ["복숭아~","오렌쥐~","샤인머스캣","사과","복숭아~","메론","참외","오렌쥐~", "복숭아~"]; push(item) : 배열의 맨 마지막으로 추가
pop() : 배열의 마지막 요소를 제거

배열에 추출/수정

arr = ["복숭아~","오렌쥐~","샤인머스캣","사과","복숭아~","메론","참외","오렌쥐~", "복숭아~"]; slice(start, end) : start ~end-1 까지의 요소를 추출
splice(idx,n,a,b,c) : 배열의 idx위치에서 n개를 삭제하고 a,b,c를 추가한다.

배열합치기 / 문자열 변환

arr = ["복숭아~","오렌쥐~","샤인머스캣","사과","복숭아~","메론","참외","오렌쥐~", "복숭아~"]; arr.concat(brr) : arr배열과 brr배열이 하나의 배열로 합쳐진다.
arr.join() : arr배열을 문자열로 변환한다.
